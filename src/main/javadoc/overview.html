This SDK contains all the tools you need to build a painlessly network connected app with java. It's primarily meant for inclusion in StackMob's Android SDK, but can be used by itself anywhere java is supported, including on the desktop.

Classes to Check Out

  • StackMobModel The base class for data models
  • StackMobUser The base class for data models representing users, with user authentication methods
  • StackMob Low level user authentication methods
  • StackMobDatastore Low level data methods
  • StackMobPush Low level push methods
